Available commands:

SPACE, N, START button and A button all begin game from Attract screen
ESC = quit application, Exit to Attract screen from game
P = toggle pause application
T = update 10x more slowly
O = single step one frame and pause
F1 = enable developer mode with debug information
F8 = reset game (only works with game running)
N = respawn player ship if ship is dead
I = spawn an asteroid
SPACE = fire a bullet
E = accelerate in the direction the ship is facing
S = turn ship left
F = turn ship right

Left Joystick = point ship and accelerate in proportion to offset in that direction
START button = respawn player ship if dead
A button = fire a bullet

F3 = kill all but first asteroid in wave (if first is dead, ends wave)
